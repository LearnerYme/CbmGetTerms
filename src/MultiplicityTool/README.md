Author: Yige Huang

## Quick start

1. copy this folder to `analysis/src` (something like this)

2. in your analysis main function, use `MultiplicityTool` instance

3. add `MultiplicityTool` into `CMakeList.txt`'s `ADD_TOOLS` as anyother modules

4. compile and run
